Due to restriction on proposed deployment platform (solace.ist.rit.edu) I decided to store images of employees and ads in the database.

Requests for '/image' endpoint:

GET /{:name} - returns the file saved under :name

POST /create - saves files submitted as form-data file fields returns a JSON array containing records files submitted. Records contain filename and ether 'success': 'ok' or 'error': error message

POST /update - creates or updates files in the database. returns a JSON array containing records files submitted. Records contain filename and ether 'success': 'ok' or 'error': error message

POST /delete/{:name} - deletes a file with :name from the database. returns a JSON containing number of records deleted
